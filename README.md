# WolframAlpha-Wikipedia-Chatbot
A smart chatbot built using Python and Streamlit that answers questions using Wolfram Alpha and Wikipedia APIs. It handles queries across various domains like math, science, and history with real-time responses and a user-friendly interface. 

# Ask Me Anything Chatbot

The "Ask Me Anything Chatbot" is an intelligent question-answering system built with Python and Streamlit, designed to deliver accurate and meaningful answers in real-time. It integrates Wolfram Alpha for computational queries and Wikipedia for general knowledge, enabling it to cover a wide range of domains including math, science, geography, history, and more.

## Key Features

- **Multi-Source Integration**: Combines Wolfram Alpha and Wikipedia APIs.
- **Natural Language Processing**: Understands and processes user questions to select the right source.
- **Fallback Mechanism**: Uses Wikipedia when Wolfram Alpha cannot provide an answer.
- **User-Friendly Interface**: Built using Streamlit for real-time interaction and responsiveness.

## Technologies Used

- **Python**: Core programming language for logic and API handling.
- **Streamlit**: Framework used to build the web-based interface.
- **Wolfram Alpha API**: Used for computational and factual queries.
- **Wikipedia API**: Used for general information and summaries.
- **Requests Library**: For making HTTP requests and fetching data.
- **dotenv Library**: For securely storing API keys and environment variables.

##  Note

Ensure `.env` is used to store sensitive keys like Wolfram Alpha API credentials.
